# 🎓 Smart Web Exam System

A modern, intelligent web-based examination platform designed to streamline online testing and assessment with real-time grading, secure proctoring, and comprehensive student analytics.

## 🌟 Features

- **Real-time Grading**: Instant feedback on student responses
- **Secure Exam Environment**: Anti-cheating measures and session management
- **Student Dashboard**: Track progress and review results
- **Teacher Dashboard**: Create exams, monitor students, generate reports
- **Multiple Question Types**: MCQ, Short Answer, Essay questions
- **Responsive Design**: Works seamlessly on desktop and tablets
- **Live Demo**: [Smart Exam System](https://smart-web-exam-system.vercel.app)

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: Vercel
- **Features**: Real-time updates, Secure authentication

## 📖 Usage

### For Students
1. Log in with your credentials
2. Select an exam from available tests
3. Complete the exam within the time limit
4. Submit and view instant results

### For Teachers
1. Create new exams with custom questions
2. Set difficulty levels and time limits
3. Monitor student progress in real-time
4. Generate performance reports and analytics
